# 🚗 Ford Customer Portal Website

A responsive and interactive customer portal website built for Ford customers to explore vehicles, manage accounts, place vehicle orders, access services, and receive support — all in one platform.

## 📌 Project Overview

The Ford Customer Portal is designed to provide Ford customers with a seamless digital experience by offering:

- Customer Login & Account Creation
- Vehicle Browsing & Ordering
- Order Tracking System
- Service & Maintenance Management
- Warranty & Claims Information
- Roadside Assistance Access
- Customer Support Portal
- Contact & Feedback System

This project focuses on frontend development using HTML, CSS, and JavaScript while integrating API-ready backend communication.

---

## ✨ Features

### 🔐 Authentication System
- Customer Login
- Account Registration
- Token-based Authentication
- Session Management using Local Storage

### 🚘 Vehicle Management
- Browse Ford Vehicle Models
- Vehicle Details & Brochures
- Vehicle Configuration Options:
  - Color Selection
  - Trim Selection
  - Engine Options

### 📦 Order Management
- Place Vehicle Orders
- View Order History
- Order Status Tracking
- Export Orders as CSV
- Filter Orders by Model & Status

### 🔧 Services Module
- Scheduled Maintenance
- Repairs & Diagnostics
- EV Services
- Extended Vehicle Plans

### 🛡 Support Section
- Owner Manuals
- Recall Information
- FAQs & Guides
- Roadside Assistance

### 📩 Contact System
- Customer Message Submission
- Feedback Form
- Customer Care Details

### 📱 Responsive Design
- Mobile Friendly UI
- Modern Dashboard Interface
- Interactive Cards & Animations

## ⚙ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Ford-Customer-Portal.git
```

2. Navigate into project folder

```bash
cd Ford-Customer-Portal
```

3. Open `index.html` in your browser
